# Portfolio Metadata Images

This folder contains images used for SEO, social media sharing, and app icons.

## Required Images

### 1. Open Graph & Twitter Card Image
- **File**: `og-image.png`
- **Size**: 1200 x 630 pixels
- **Purpose**: Used when sharing on social media (Facebook, Twitter, LinkedIn, etc.)
- **Design Tips**:
  - Include your name "Rahul Arora"
  - Add tagline "Full Stack Developer"
  - Use your brand colors
  - Keep text readable
  - Test how it looks on different platforms

### 2. Favicon
- **File**: `favicon.ico`
- **Size**: 32 x 32 pixels (multi-size ICO file recommended)
- **Purpose**: Browser tab icon

### 3. App Icons
- **File**: `icon-192.png`
- **Size**: 192 x 192 pixels
- **Purpose**: PWA icon, Android home screen

- **File**: `icon-512.png`
- **Size**: 512 x 512 pixels
- **Purpose**: PWA icon, Android splash screen

### 4. Apple Touch Icon
- **File**: `apple-icon.png`
- **Size**: 180 x 180 pixels
- **Purpose**: iOS home screen icon, Safari pinned tab

## Design Recommendations

1. **Use a consistent brand identity** across all images
2. **Keep it simple** - Your initials "RA" or a logo work well
3. **Use high contrast** for visibility on different backgrounds
4. **Test on different devices** to ensure clarity
